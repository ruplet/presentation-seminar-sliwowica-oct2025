# presentation-seminar-sliwowica-oct2025
Files for presentation about Lean 4 formalization of bounded arithmetic on Sliwowica seminar at the University of Warsaw.

Sliwowica seminar is a faculty-focused seminar, lead by the Programming Languages and Semantics research group. The format of the presentation was 90 minutes -- I have presented an extended version of my [AITP'25 presentation](https://github.com/ruplet/presentation-aitp2025), focusing less on the foundations (deduction system, what arithmetic is etc.), and more on the meat -- how to formalize this in Lean 4.
